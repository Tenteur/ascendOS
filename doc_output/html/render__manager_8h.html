Detailed Description

This render_manager contains everything needed to manager the renderer.

Note
Need a lot of work, will not be operationnal for long time. This script aim to simplify the rendering and keeping/destroying textures by managing them automatically. Once this script is complete, should disallow the use of raw SDL2 rendering.

signed int layer,
SDL_Surface * surfacePtr,
const int x,
const int y )

Add a surface to draw to screen. Add a texture to draw and return the id of this texture relative to scene.

Note
The surface gets converted to a texture before being drawn.
Parameters
sceneIDThe ID of the scene where the texture is drawn (1 background, 2 middle, 3 foreground).
layerThe layer to draw the surface, lower layer gets drawn first (0->1->2) if two surface have the same layer but overlap, the first to be drawn is based on the ID the texute gets.
surfacePtrThe pointer to the surface to add.
xThe x coordinate where the surface gets drawn relative to screen (absolute so).
yThe y coordinate where the surface gets drawn relative to screen (absolute so).
Returns
ID of the surface that got added, -1 if error occured, ID > 0 if success
