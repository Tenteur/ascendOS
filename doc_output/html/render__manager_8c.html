Add a texture to draw to screen. Add a texture to draw and return the id of this item relative to scene.

Note
One of texturePtr and surfacePtr should be NULL, the one that is not NULL is the one that will be drawn.
Parameters
layerThe ID of the scene where the texture is drawn (0 background, 1 middle, 2 foreground).
sublayerThe sublayer to draw the texture, lower sublayer gets drawn first (0->1->2) if two texute have the same sublayer but overlap, the first to be drawn is based on the ID the texute gets.
texturePtrThe pointer to the texture to add, can be NULL (check note).
surfacePtrThe pointer to the surface to add, can be NULL (check note).
xThe x coordinate where the texture gets drawn relative to screen (absolute so).
yThe y coordinate where the texture gets drawn relative to screen (absolute so).
widthThe width of the item.
heightThe height of the item.
Returns
itemID > 0 if success, 0 if failure.
